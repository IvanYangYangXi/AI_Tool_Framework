## 项目概述
DCC工具框架是一个统一的插件管理平台，旨在为各种数字内容创作(DCC)软件和游戏引擎提供标准化的工具开发和管理解决方案。

## 原始设计规划

*以下为原始设计文档内容，保留作为历史参考*

### DCC插件接口规范 ✅
- 定义了统一的插件接口标准
- 支持多种DCC软件(Maya, 3ds Max, Blender等)
- 提供参数验证和错误处理机制

### UE引擎插件接口规范 ✅
- 专门针对Unreal Engine的插件标准
- 支持UE5.0-5.4版本
- 包含UE特有的编辑器集成功能

### 插件依赖管理系统 📦
*（规划功能，当前版本未实现）*

- 检测版本兼容性和冲突
- 提供智能安装顺序建议

### 插件市场功能 📦
*（规划功能，当前版本未实现）*

## 技术特性

### 架构优势
- **微内核设计**: 核心系统轻量，功能通过插件扩展
- **统一接口标准**: 所有插件遵循相同的设计规范
- **跨平台兼容**: 支持Windows、macOS、Linux

### 开发者友好
- **清晰的API文档**: 详细的接口说明和使用示例
- **完善的错误处理**: 友好的错误提示和日志记录
- **灵活的配置系统**: 支持JSON格式的插件配置

## 支持的软件平台

### DCC软件
- Autodesk Maya (2022-2025)
- Autodesk 3ds Max (2021-2024)  
- Blender (3.0-4.2)
- (预留扩展) Houdini, Cinema 4D, Nuke

### 游戏引擎
- Unreal Engine (5.0-5.4)
- (预留扩展) Unity, Godot

## 当前项目结构

```
src/
├── core/                           # 核心组件 ✅
│   ├── dcc_plugin_interface.py    # DCC插件接口
│   ├── ue_plugin_interface.py     # UE插件接口
│   ├── plugin_manager.py          # 插件管理器
│   ├── config_manager.py          # 配置管理器
│   ├── dynamic_loader.py          # 动态加载器
│   └── permission_system.py       # 权限系统
│
├── gui/                            # 图形界面 ✅
│   └── lightweight_manager.py     # 轻量级管理器
│
├── plugins/                        # 插件目录 ✅
│   ├── dcc/                       # DCC工具插件
│   │   ├── maya/example_print_selection/
│   │   ├── max/example_print_selection/
│   │   └── blender/example_print_selection/
│   └── ue/                        # UE引擎插件
│       └── example_print_selection/
│
└── main.py                        # 主入口
```
