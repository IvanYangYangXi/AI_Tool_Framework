# Unreal Engine资产优化工具

## 描述
这是一个专业的Unreal Engine资产优化插件，基于UE工具框架标准接口开发。该工具提供全面的资产优化功能，包括纹理压缩、网格LOD生成、材质实例化等。

## 功能特性

### 核心功能
- **纹理压缩优化**: 智能压缩纹理资源，平衡质量和文件大小
- **网格LOD生成**: 自动生成多级细节模型
- **材质实例化**: 将复杂材质转换为可配置实例
- **碰撞体优化**: 优化物理碰撞形状
- **资产分析**: 详细的资产大小和性能分析
- **报告生成**: 自动生成优化报告

### 技术特点
- 基于标准UE插件接口规范
- 支持参数验证和错误处理
- 提供详细的处理报告和统计信息
- 可配置的优化参数
- 支持批量处理多种资产类型

## 安装方法

### 方法一：框架自动安装
1. 在UE工具框架中搜索"UEAssetOptimizer"
2. 点击安装按钮
3. 插件将自动部署到正确位置

### 方法二：手动安装
1. 将`plugin.py`文件复制到UE项目的`Plugins`目录
2. 在UE编辑器中重启项目启用插件

## 使用方法

### 基础使用
1. 在UE编辑器中选择要优化的资产
2. 运行插件
3. 设置优化参数：
   - **纹理质量**: 控制纹理压缩程度 (1-100)
   - **生成LOD**: 是否为网格生成细节级别
   - **材质实例化**: 创建可配置的材质实例
   - **压缩纹理**: 应用纹理压缩算法
   - **优化碰撞**: 优化物理碰撞形状

### 高级使用
```python
# 通过代码调用
optimizer = UEAssetOptimizer()
result = optimizer.execute(
    texture_quality=80,           # 纹理质量80%
    generate_lods=True,          # 生成LOD
    create_material_instances=True, # 创建材质实例
    compress_textures=True,      # 压缩纹理
    optimize_collision=True,     # 优化碰撞
    output_report=True          # 生成报告
)
```

## 参数说明

| 参数名 | 类型 | 默认值 | 范围 | 描述 |
|--------|------|--------|------|------|
| texture_quality | integer | 75 | 1-100 | 纹理压缩质量 |
| generate_lods | boolean | True | - | 生成网格LOD |
| create_material_instances | boolean | True | - | 创建材质实例 |
| compress_textures | boolean | True | - | 压缩纹理 |
| optimize_collision | boolean | True | - | 优化碰撞体 |
| output_report | boolean | True | - | 输出优化报告 |

## 支持的资产类型

- **StaticMesh**: 静态网格
- **SkeletalMesh**: 骨骼网格  
- **Texture2D**: 二维纹理
- **Material**: 材质
- **MaterialInstance**: 材质实例

## 输出报告

插件会生成详细的优化报告，包含以下信息：
- 处理时间和资产数量
- 成功/失败统计
- 各类资产的优化详情
- 压缩比率和节省空间
- 性能改进建议

## 兼容性

- **UE版本**: 5.0 - 5.4
- **项目类型**: C++项目
- **平台**: Windows, macOS, Linux
- **Python版本**: 3.7+ (UE内置)

## 性能提示

- 大规模资产优化可能需要较长时间
- 建议在处理前备份原始资产
- 可以分批处理不同类型资产
- 质量设置越低，处理速度越快但质量越差

## 故障排除

### 常见问题
1. **插件无法加载**: 检查UE版本兼容性
2. **优化无效果**: 确认选择了正确的资产类型
3. **性能慢**: 降低质量设置或分批处理
4. **内存不足**: 减少同时处理的资产数量

### 日志查看
插件会在UE输出日志中输出详细的操作日志，可用于诊断问题。
