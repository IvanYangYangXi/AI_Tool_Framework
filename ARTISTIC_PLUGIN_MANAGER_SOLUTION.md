## 📱 使用流程

### 美术人员使用步骤
1. **启动程序**: 双击启动器或exe文件
2. **选择插件**: 在界面中浏览和选择需要的工具
3. **配置参数**: 通过可视化界面设置运行参数
4. **执行工具**: 点击运行按钮开始处理
5. **查看结果**: 在日志区域查看执行结果

### 管理员部署步骤
1. **环境准备**: 确保Python环境正常
2. **运行打包**: 执行`python build_exe.py`
3. **分发程序**: 将生成的exe文件分发给用户
4. **技术支持**: 提供使用说明和帮助文档
